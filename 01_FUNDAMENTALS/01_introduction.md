# Networking and Types of Networks

## Definition

A **network** is a structured system that interconnects two or more devices, enabling the sharing of data, resources, and services. Networks underpin modern communication, supporting everything from file sharing to internet access.

## Communication Types

- **Unicast:**  
    One-to-one communication, where data is sent from a single sender to a single receiver.  
    *Example:* Sending an email to a specific recipient.

- **Multicast:**  
    One-to-many communication, but only to a selected group of recipients.  
    *Example:* Streaming a live lecture to registered students.

- **Broadcast:**  
    One-to-all communication, where data is transmitted to all devices within the network segment.  
    *Example:* Announcing a message over a school public address (PA) system to all students.

## Network Types by Geographical Scope

- **Local Area Network (LAN):**  
    Connects devices within a confined area, such as a home, office, or campus. LANs offer high data transfer rates and low latency.  
    *Example:* Office computers connected via Ethernet.

- **Wide Area Network (WAN):**  
    Spans large geographical areas, connecting multiple LANs, often across cities, countries, or continents. WANs typically use leased telecommunication lines.  
    *Example:* The Internet, or a corporate network linking branch offices worldwide.

## Communication Modes

- **Simplex:**  
    Data flows in only one direction; the sender cannot receive, and the receiver cannot send.  
    *Example:* Television or radio broadcasting.

- **Half Duplex:**  
    Data transmission is bidirectional, but only one device can send at a time.  
    *Example:* Walkie-talkies, traditional Ethernet hubs.

- **Full Duplex:**  
    Data transmission is bidirectional and simultaneous, allowing both devices to send and receive at the same time.  
    *Example:* Modern telephones, network switches.

---

**General Notes:**
- Networks can be classified by size, topology, and purpose.
- Efficient networking enhances collaboration, resource sharing, and communication.
- Security and management are critical considerations in network design.
- Other network types include MAN (Metropolitan Area Network) and PAN (Personal Area Network).
- Network protocols (e.g., TCP/IP) define rules for data exchange.
- Wired and wireless technologies both play significant roles in networking.
